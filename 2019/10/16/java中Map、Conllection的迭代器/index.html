java中Map、Conllection的迭代器

迭代器Iterator

Iterator是java.lang包中自带的接口，被Collection继承。

Iterator只能向后遍历，不能向前遍历。

Iterator常用方法

iterator()：要求容器返回一个Iterator对象

next()：第一次调用时返回第一个元素，以后调用发那会下一个元素。

hasNext()：检查序列中是否还有元素。

remove()：将返回的元素删除。

List list;
Iterator<obj> iterator = list.iterator();
while(iterator.hasNext())
    {
    System.out.print(iterator.next();)
    }

迭代器ListIterator

继承于Iterator，被List继承，Set不能用。可以双向遍历，可以插入和删除元素

next();

previous(); 向前遍历

hasNext();

hasPrevious(); 判断是否有前一个

add(); 添加一个元素

nextIndex(); 前一个元素的索引值，即下标

previousIndex(); 后一个元素的索引值，即下标

*set(); * 修改当前元素的值

List list = new ArrayList();

ListIterator<obj> li = list.listIterator();
while(li.hasNext())//+迭代输出
while(li.hasPrevious())//+迭代输出

set集合

set迭代基本和list一样使用

另外，set中存放 map需要借助Map.Entry<K,V>

Map遍历

需要借助Entry<K, V>

先用 map.entrySet()转化成Set集合里面存着Entry<K, V>

然后遍历set集合

具体实现：

Map<int,String> map1 = new HashMap<>();
/***lue***/
Set<Entry<int,String>> ss = map1.entrySet();
for(Iterator<Entry<int,String>> iterator = ss.iterator();iter.hasNext();){
    Entry<int,String> temp = iterator.next();
    System.out.println(temp.getKey()+temp.getValue());
}
