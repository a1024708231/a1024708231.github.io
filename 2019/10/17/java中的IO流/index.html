    分类：

    按照流的方向：

    输入流 输出流

    按照流的处理数据：

    字符流 字节流

    按照功能分类：

    节点流 处理流

    有FIle 和 ByteArray 的都是节点流 即文件和数组，直接数据源

    一、File

    初始化

    File src = new File(“D:\blog\sources\java中的IO流.md”);

    路径说明：

    绝对路径：”D:\blog\sources\java中的IO流.md”

    相对路径：”sources\java中的IO流.md”

    父路径：”D:\blog\sources”

    ————————————————

    getName（） 获得文件名

    getPath（） new 的时候给的是相对路径就返回相对路径，是绝对路径就返回绝对路径

    getAbsoulutePath（） 返回绝对路径

    getParent（） 返回父路径，如果没有，返回null

    getParentFile（） 返回父对象

    exists（） 判断是否存在

    isFile（） 判断是否文件

    isDirectory（） 判断是否文件夹

    length（） 获取文件的长度，即文件的大小，如果是文件夹或者不存在则为0

    createNewFile（） 创建文件，不能创建文件夹

    delete（） 删除 

    File src = new File("D:/blog/io.txt");
boolean flag = src.createNewFile();//不存在则返回true创建成功，存在则返回false

    文件夹

    mkdir（） 上级目录存在才可以创建 返回true或false

    mkdirs（） 上级目录存不存在都可以创建 返回true或false （推荐使用mkdirs）

    File dir = new File();
boolean flag = dir.mkdirs();

    list（） 返回下级文件的名称数组 String[]

    listFiles（） 返回下级文件对象数组 File[]

    listRoots（） 列出所有盘符， C:\ D:\ E:\ F:\

    遍历文件夹

    public Static void main(){
    File src = new File("D:\blog");
    printName(src);
}
public Static void printName(File src){
    System.out.println(src.getName());
    if(src==null||!src.exists()){
        return;
    }else if(src.isDirectory()){
        for(File s:src.listFiles()){
            printName(s);     //递归调用自己
        }
    }
}

    释放资源：

    InputStream

    实现了Closeable接口，可以通知操作系统关闭资源

    close（）

    read（） 一个字节一个字节的读 返回读到的字节

    read（byte[] b） 一段一段的读 从输入流中读一些字节，储存到缓冲数组byte[] b 中

    实现了Closeable接口，可以通知操作系统关闭资源

    OutputStream

    实现了Closeable接口，可以通知操作系统关闭资源。

    close（）

    实现了Flushable， 可以用

    flush（）刷新

    write(byte[] b) 将b中的字节写入此输出流

    write（byte[] b, int off, int len） 从指定的字节数组b的一部分写入，从off开始读len个数

    实现Readable接口，可读的

    read（char() c） 将字符读入char() c 中 返回长度

    Writer

    Appendable接口，可追加的写

    处理流

    处理流即缓冲流，内部维持了一个数组缓冲数据，提高效率。

    使用装饰模式

    释放最外层的处理流，内部的字节流会自动释放。手动关需要从里关到外

    InputStream is = null;
try{
    is = new BufferedInputStream(new FileInputStream(src));
    byte[] flush = new byte [1024];//缓冲容器
    int len = -1;//返回的接受长度
    while（(len=is.read(flush))!=-1）{
        //字节数组--》字符串（解码）
        String str = new String(flush,0,len);
        System.out.println(str);
    }
}catch(FlieNotFoundException e){
    e.printStackTrace();
}catch(IOException e){
    e.printStackTrace();
}finally{
    //释放资源
}
